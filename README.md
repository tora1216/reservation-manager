# 予約リスト

お店の予約情報・予約開始日を管理するアプリ。

## 機能

- お店の追加・編集・削除（削除時は確認ダイアログあり）
- カテゴリ管理（絵文字付き、上下矢印で並び替え、編集・削除対応）
  - カテゴリ削除時は対象のお店を「その他」へ自動移動
- 予約開始スケジュール管理（毎日・毎週・毎月）と残り日数バッジ表示
- 予約対象日の自動計算・表示（何ヶ月/週/日先の予約かを設定可能）
- カテゴリフィルター・並べ替え（開放順／登録順／名前順）
- ダークモード対応
- iPhoneホーム画面への追加対応（PWA）

## 開発

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) で確認。
